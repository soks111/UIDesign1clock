# AGENTS.md

## Project Intent

This is a "clock that is not quite a clock" project.

It is preparation material for a workshop:
https://docs.google.com/presentation/d/1XnEmxPVkdmQoYJaCU591APvBd3416kknDMXVLrHndHs/edit?usp=sharing

Keep the implementation small, direct, and playful. The core constraint is part of the work:

- Use only `new Date()` to read the current time.
- Use only `setInterval()` to schedule updates.
- Do not add frameworks, build steps, libraries, or time/date helper packages.
- Prefer plain HTML, CSS, and JavaScript in `index.html` and `index-*.html`.
- Use the original `index.html` as the baseline for structure and style.
- Avoid `main` and extra wrapper elements unless there is a clear reason.
- Default to `color: white` and `background: black`.
- Keep changes understandable at a glance.
- Prefer workshop-friendly examples that are easy to explain, copy, and modify.

When improving the project, preserve the constraint before adding polish.
